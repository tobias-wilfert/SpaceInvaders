Detailed Description


Inspiration: https://www.tutorialspoint.com/Explain-Cplusplus-Singleton-design-pattern
A Singleton implementation of a Transformation from [-4,4]x[-3,3] to 2D game pixels
Invariant: windowWidth and windowHeight > 0
Usage:
std::shared_ptr<Transformation> Transformation::instance = nullptr;
As Non-const static member must be initialized out of line
std::shared_ptr<Transformation> t = Transformation::initialize(width,height);
After that this is possible
std::shared_ptr<Transformation> t2 = Transformation::getInstance();
