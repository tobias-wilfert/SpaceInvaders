Checks that:
1.) co1 and co2 are both not destroyed
2.) co1 and co2 are not dead -> throws an logic_error
3.) co1 and co2 collided
If yes, the 2 objects collided decreasing each others HealthPoint with there AttackPoints
If one of the objects is now dead set it to destroyed
